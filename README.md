# Proyecto Manejo Buses

## Descripción

Proyecto Manejo Buses es un sistema web académico desarrollado en PHP Laravel para la gestión de transporte interprovincial. El sistema permite administrar cooperativas, buses, rutas, horarios, usuarios, boletos y selección visual de asientos.

El proyecto busca organizar los procesos principales de una empresa o cooperativa de transporte, facilitando el registro de información, la administración de rutas y la gestión de boletos desde una aplicación web.

## Objetivo general

Desarrollar una aplicación web en Laravel para la gestión de buses y servicios de transporte, permitiendo administrar cooperativas, flota vehicular, rutas, horarios, usuarios y boletos.

## Funcionalidades principales

- Gestión de cooperativas de transporte.
- Registro y administración de buses.
- Gestión de rutas y horarios.
- Gestión de usuarios.
- Búsqueda de rutas disponibles.
- Selección visual de asientos.
- Registro y control de boletos.
- Administración de información mediante panel web.
